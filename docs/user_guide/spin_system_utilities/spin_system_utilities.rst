=====================
Spin System Utilities
=====================

Czjzek and Extended Czjzek distributions
----------------------------------------

``mrsimulator`` offers the capability to efficently create a large number of spin systems with
modeling amorphous materials using the **Czjzek** and **Extended Czjzek** distributions. Click
on the links below to view pages documenting how to set up and use the **Cjzjek** and
**Extended Czjzek** distributions.
